# CRISPRbact Streamlit Interface

A browser-based interface for CRISPRbact.

This project provides a Streamlit web interface for the CRISPRbact toolkit developed by the Bikard Lab at Institut Pasteur.

Features:
- Library Design
- Library Mapping
- Add-on Library

The underlying CRISPRbact algorithms and methods are unchanged. This project focuses on improving accessibility through a graphical web interface.
